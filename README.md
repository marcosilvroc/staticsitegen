# Static Site Generator
This project is a static site generator from a course on [Boot.dev](https://www.boot.dev), a live demo of the current project can be access [here](https://marcosilvroc.github.io/staticsitegen/).

If you want to run this project locally, download the files and execute **main.sh** script, then you can access in your browser via **http://localhost:8888**.

## How it works
For each markdown file located inside the content folder, the generator transforms it into a HTML file and writes them in a public directory (if locally) then the built-in python HTTP server is executed and the site can be accessed in `http://localhost:8888`.
